# 🎮 Tic Tac Toe AI (PyScript)

Project ini adalah implementasi game **Tic Tac Toe** berbasis web yang
menggunakan **Python (PyScript)** sebagai logika utama, serta dilengkapi
dengan AI menggunakan algoritma:

-   Minimax
-   Alpha-Beta Pruning

------------------------------------------------------------------------

## ✨ Fitur Utama

-   🎯 Bermain melawan AI
-   🧠 Pilihan algoritma (Minimax / Alpha-Beta)
-   ⚙️ Pengaturan depth pencarian
-   📊 Statistik performa AI (node, waktu, pruning, dll)
-   🎨 Tampilan UI modern dan responsif

------------------------------------------------------------------------

## 📁 Struktur Project

    📦 tic-tac-toe-ai
    ├── index.html
    ├── style.css
    ├── app.py
    └── README.md

## 🚀 Cara Menjalankan (Localhost)

Ikuti langkah berikut untuk menjalankan project di komputer kamu:

1.  Buka folder project
2.  Klik kanan pada folder/file project
3.  Pilih **Open in Terminal**
4.  Jalankan perintah berikut:

```{=html}
    py -m http.server 8000
```
5.  Buka browser dan akses:

```{=html}
    http://localhost:8000/
```
6.  🎉 Game siap dimainkan!

------------------------------------------------------------------------

## 🎮 Cara Bermain

1.  Pilih:
    -   Algoritma AI
    -   Depth pencarian
    -   Simbol (X atau O)
2.  Klik tombol **Game Baru**
3.  Mainkan game melawan AI
4.  Perhatikan statistik performa AI di panel samping

------------------------------------------------------------------------

## 🧠 Penjelasan Singkat AI

-   **Minimax**: Algoritma yang mengevaluasi semua kemungkinan langkah
    untuk mencari hasil terbaik.
-   **Alpha-Beta Pruning**: Optimasi dari minimax yang mengurangi jumlah
    node yang dievaluasi sehingga lebih cepat.
